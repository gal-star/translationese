Code Documentation
==================
This page describes the main code elements in the translationese module.

The Analysis class
----------------------
The Analysis class, given a text, provides helper functions for text analysis using NLTK library. The attributes then operates those functions. The estimation conducted by the attributes, sets as feature input to the SVM classification. Therefore, the classification is done based on the attributes estimation.

Unit Tests
----------
This work was written unit test driven. Unit tests main idea is to assure that given certain input, the desired result would be as outputed by the code.
